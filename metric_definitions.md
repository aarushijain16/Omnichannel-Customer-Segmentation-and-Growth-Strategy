# Metric Definitions

This document defines key engineered metrics used across the project.



# Engagement Index

Composite score derived from:

- Visit frequency

- Channel interactions

- Category diversity

Higher values indicate deeper brand interaction.



# Loyalty Score

Derived from:

- Purchase consistency

- Repeat behavior

- Recency proxy

Used to distinguish habitual vs occasional customers.



# Churn Risk Proxy

Relative risk score based on:

- Declining engagement

- Reduced frequency

- Channel disengagement

Used directionally, not probabilistically.



# CLV Proxy

Estimated using:

- Average order value

- Purchase frequency

- Engagement weighting

Represents **relative lifetime value**, not monetary forecasting.


## Online Purchase Ratio

- Proportion of customer activity occurring online vs in-store.

- Used to inform omnichannel strategy and UX decisions.
